arbitroom-account2

Документация для php-бекенда по верстке arbitroom-account2

Принципы

1. Это в первую очередь верстка, а не готовый фронт. Тут нет взаимодествия с бэкендом. Оно может иммитироваться с помощью фейковых данных и скриптов.

Данные находятся в папке /_fake в виде json-файлов.

Фейковые скрипты находятся прямо в html и помечены коментарием <!-- fake script -->

2. BEM-подобный синтаксис

3. Данные для компоненты передаются через data-аттрибуты

4. Компоненты могут выбрасывать CustomEvent

5. Компоненты в которых меняется значение, содержат в себе скрытый input. На него можно повесить обработчик события change
