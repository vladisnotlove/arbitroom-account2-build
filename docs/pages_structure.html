Отображает массив сот из json в атрубуте data-source

При изменение data-source соты перерендориваются

Формат одной соты:

            
{
	id: number;
	content: {
		pretitle: string;
		status: string;
		name: string;
		turn: string;
		turnLabel: string;
	};
	theme?:
		| "A1"
		| "A2"
		| "A3"
		| "B1"
		| "B1-F"
		| "B2"
		| "B2-F"
		| "B3"
		| "B3-F"
		| "C1"
		| "C1-F"
		| "C2"
		| "C2-F"
		| "C3"
		| "C3-F"
		| "D1"
		| "D2"
		| "D3"
		| "none"
		| "base";
	backgroundSrc?: string;
	detailsUrl?: string;
	chatUrl?: string;
}

Также выбрасывает событие hex-select c сотой в detail.data

Пример использования:

            
hexs.addEventListener("hex-select", (e) => {
	console.log(e.detail.data);
});
